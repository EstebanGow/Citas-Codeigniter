Aplicación para citas Medicas con el framework PHP Codeigniter

- Pasos para Ejecutar Aplicación:

1. Ejecutar un Servidor de PHP en la maquina.
2. Ejecutar un Administrador de Bases de Datos.
3. En la carpeta database se encuentra un archivo .sql, este se debe importar en phpmyAdmin una vez hecho esto se creara una base de datos llamada medicaldb.
4. pasar la carpeta que contiene todos los archivos al disco local C a la carpeta que usar por defecto el servido Apache que se este utilizando.
5.Acceder a la Ruta por defecto del localhost y agregar el nombre de la carpeta donde se agregaron los archivos descargados. 

NOTA: la url por defecto es: http://localhost/Citas/   
En caso de que su Servido este trabajando en un puerto diferente se debe configurar el 
base_url en la carpeta config del proyecto.
